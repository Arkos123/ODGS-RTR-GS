## 三、desify_and_prune 剪枝逻辑

定义在 [gaussian_model.py:L1089-L1108](file:///home/huangpengyue/projects/RTR-GS/scene/gaussian_model.py#L1089-L1108)：

```python
def densify_and_prune(self, max_grad, min_opacity, extent, max_screen_size, weights_threshold=1e-4):
    grads = self.xyz_gradient_accum / self.denom
    grads[grads.isnan()] = 0.0

    self.densify_and_clone(grads, max_grad, extent)
    self.densify_and_split(grads, max_grad, extent)

    prune_mask = (self.get_opacity < min_opacity).squeeze()
    weight_mask = self.weights_accum[:, 0] < weights_threshold
    prune_mask = torch.logical_or(weight_mask, prune_mask)
    # print("weights_accum:", weight_mask.sum().item(), "/", self.get_xyz.shape[0])
    if max_screen_size:
        big_points_vs = self.max_radii2D > max_screen_size
        big_points_ws = self.get_scaling.max(dim=1).values > 0.1 * extent
        prune_mask = torch.logical_or(torch.logical_or(prune_mask, big_points_vs), big_points_ws)

    self.prune_points(prune_mask)
    self.weights_accum.data[:] = 0.0

    torch.cuda.empty_cache()
```

**剪枝条件（满足任一即被移除）**：
1. **低透明度**：`opacity < min_opacity`（即 `< 0.005`）
2. **低权重累积**：`weights_accum < 1e-4`
3. **过大高斯**：`max_radii2D > max_screen_size`（仅在 `max_screen_size` 不为 `None` 时启用）
4. **世界空间过大**：`scaling.max() > 0.1 * extent`

## 五、不透明度重置逻辑

### reset_opacity — 标准重置 ([gaussian_model.py:L611-L614](file:///home/huangpengyue/projects/RTR-GS/scene/gaussian_model.py#L611-L614))

```python
def reset_opacity(self):
    opacities_new = inverse_sigmoid(torch.min(self.get_opacity, torch.ones_like(self.get_opacity) * 0.01))
    optimizable_tensors = self.replace_tensor_to_optimizer(opacities_new, "opacity")
    self._opacity = optimizable_tensors["opacity"]
```

将所有高斯的不透明度 **钳制到最高 0.01**（即 `min(current, 0.01)`），防止不透明度在训练过程中过高导致高斯无法被剪枝。

### reset_opacity1 — 提高到 0.9 ([gaussian_model.py:L617-L627](file:///home/huangpengyue/projects/RTR-GS/scene/gaussian_model.py#L617-L627))

```python
def reset_opacity1(self, exclusive_msk = None):
    RESET_V = 0.9
    opacity_old = self.get_opacity
    o_msk = (opacity_old > RESET_V).flatten()
    if exclusive_msk is not None:
        o_msk = torch.logical_or(o_msk, exclusive_msk)
    opacities_new = torch.ones_like(opacity_old)*inverse_sigmoid(torch.tensor([RESET_V]).cuda())
    opacities_new[o_msk] = self._opacity[o_msk]
    optimizable_tensors = self.replace_tensor_to_optimizer(opacities_new, "opacity")
    if "opacity" not in optimizable_tensors: return
    self._opacity = optimizable_tensors["opacity"]
```

将所有高斯的不透明度 **提高到 0.9**，但排除掉原本已经 `> 0.9` 的高斯以及 `exclusive_msk`（外部区域）中的高斯。用于法向传播阶段的健壮性增强。

### reset_refl — 反射强度重置 ([gaussian_model.py:L632-L638](file:///home/huangpengyue/projects/RTR-GS/scene/gaussian_model.py#L632-L638))

伴随 `reset_opacity()` 触发，将反射强度钳制到最高 0.01。
